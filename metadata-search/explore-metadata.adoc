= 워크스페이스 파일 메타데이터 탐색 및 내보내기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
AIDE 소프트웨어 전용 배포를 사용하는 AIDE 사용자는 AI Data Engine(AIDE) Console의 Metadata Engine 옵션을 사용하여 Metadata Engine을 통해 파일 메타데이터를 검색하고 필터링할 수 있습니다. 표준 필터를 적용하거나 사용자 지정 태그로 검색할 수 있으며, 결과를 CSV 파일로 내보내 오프라인 분석 또는 다운스트림 용도로 활용할 수 있습니다.


NOTE: 전체 AIDE 라이센스 없이 AIDE 소프트웨어 전용 배포에서는 Metadata Engine 기능에 대한 전체 액세스 권한이 있지만 데이터 컬렉션이나 가드레일을 생성하거나 관리할 수 없습니다. link:../faq-aide.html["배포 및 라이선스 옵션에 대해 알아보십시오"].

.시작하기 전에
* AIDE Console에서 _스토리지 관리자_, _데이터 엔지니어_ 또는 _데이터 과학자_ 권한이 있어야 하며 하나 이상의 워크스페이스에 대한 액세스 권한이 필요합니다.
* 스토리지 관리자는 다음과 같은 권한을 가지고 있습니다:
+
** ONTAP System Manager에서 하나 이상의 워크스페이스를 생성했습니다.
** link:../install-setup/configure-oidc.html["OpenID Connect(OIDC)"]을 사용하여 관련 작업 공간에 대한 사용자 또는 그룹 액세스 권한을 할당했습니다.


* 워크스페이스의 메타데이터 추출이 완료되었으며 워크스페이스는 `Ready` 상태입니다.




== AIDE Console에 Sign in

.단계
. 브라우저에서 AIDE Console URL로 이동하세요.
+
`\https://<cluster_management_ip>/console`

. 조직의 OIDC 공급자를 통해 인증합니다.
. 왼쪽 탐색 패널에 *Data Explorer*가 있는지 확인하십시오.
+

NOTE: 전체 AIDE 라이선스 없이 AIDE 소프트웨어만 배포하는 경우, *Data Explorer*가 기본 시작 페이지입니다.



.결과
AIDE Console에 로그인되어 있으며 액세스 권한이 부여된 작업 공간을 볼 수 있습니다.



== Data Explorer에서 작업 공간 열기

.단계
. 왼쪽 탐색 패널에서 *Data Explorer > Workspaces*를 선택합니다.
. 사용 가능한 작업 영역 목록을 검토합니다.
. 탐색하려는 작업 공간을 선택하세요.
. 해당 작업 공간의 파일 메타데이터 검색 및 필터 페이지를 열려면 *탐색*을 선택하십시오.


.결과
선택한 작업 영역으로 범위가 지정된 Data Explorer 검색 페이지가 열립니다.



== 파일 메타데이터 검색 및 필터링

.단계
. Data Explorer 검색 페이지에서 사용 가능한 필터를 조합하여 결과를 세분화할 수 있습니다.
+
** 파일 이름이나 메타데이터 속성을 일치시키려면 검색창에 검색어를 입력하세요.
+

NOTE: 검색 쿼리는 대소문자를 구분합니다.

** 표준 메타데이터 필터(예: 파일 형식, 크기 또는 날짜)를 적용합니다.
+

NOTE: 데이터 탐색기에서 *분류* 필터를 사용할 수 없습니다.





.결과
이 표에는 검색어 또는 적용된 필터와 일치하는 모든 파일이 표시됩니다.



== 사용자 지정 태그 생성 및 사용

.단계
. 사용자 지정 태그를 만들려면 Data Explorer 검색 페이지로 이동하여 *고급 필터*를 선택하세요.
. *새 태그 추가*를 선택합니다.
. 표준 필터에 포함되지 않은 속성 조합으로 검색 결과를 필터링하려면 하나 이상의 사용자 지정 태그를 추가하세요.
. 검색 결과에 사용자 지정 태그 필터를 적용하려면 *필터*를 선택하세요.
. 파일 목록 표에 표시된 검색 결과를 검토하십시오.


.결과
이 표에는 사용자 지정 태그 항목과 일치하는 모든 파일이 표시됩니다.



== 검색 결과를 CSV로 다운로드

.단계
. 검색 결과 표 위에서 image:icon_download_file.png["파일 다운로드 아이콘"]을(를) 선택합니다.
+
다운로드가 완료될 때까지 다운로드 진행 상황을 알려주는 메시지가 표시됩니다.

. 결과 테이블 위의 다운로드 진행률 표시기를 모니터링합니다. 대용량 다운로드의 경우 일괄 파일 다운로드 진행률 또는 처리 중인 CSV의 행 수가 표시됩니다.
. 선택 사항: 다운로드가 완료되기 전에 중지하려면 *다운로드 취소*를 클릭합니다.


.결과
하나 이상의 CSV 파일이 로컬 시스템으로 다운로드됩니다. 각 파일에는 다운로드 시점에 검색 결과 테이블에 표시되는 파일 이름과 메타데이터 열이 포함되어 있습니다.
