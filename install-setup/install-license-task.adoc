= AFX 시스템에 AIDE 라이센스를 설치합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp 라이선스를 설치해야 AI Data Engine(AIDE)의 모든 기능을 사용할 수 있습니다. ONTAP 클러스터 관리자는 ONTAP System Manager를 사용하여 라이선스를 관리할 수 있습니다.



== AI Data Engine 라이센스 준비

라이선스는 하나 이상의 소프트웨어 사용 권한에 대한 기록입니다. 모든 AFX 라이선스는 NetApp 라이선스 파일(NLF) 형태로 제공되며, 이 단일 파일로 여러 기능을 사용할 수 있습니다. AIDE를 지원하기 위해 AFX 스토리지 시스템에 라이선스를 설치하기 전에 고려해야 할 몇 가지 사항이 있습니다.



=== 라이센스 유형

AFX 스토리지 시스템을 통해 AIDE를 사용하기 시작하는 데 필요한 두 가지 주요 라이센스 유형이 있습니다.

.ONTAP One 라이센스
ONTAP One 라이센스가 있는 경우 Metadata Engine Basic 라이센스가 활성화됩니다. 이를 통해 AIDE 작동의 필수 기반을 제공하는 Metadata Engine 기능에 액세스할 수 있습니다.

.AI Data Engine 프리미엄 라이센스
벡터화, 거버넌스 가드레일 및 추론과 같은 AI 기능을 활성화하고 사용하려면 AIDE 프리미엄 서비스 라이선스에 대한 액세스 권한을 획득하고 설치해야 합니다. 라이선스에는 GPU 수와 만료 날짜가 포함됩니다.


NOTE: AIDE의 전체 기능은 특정 NetApp DCN 구축 환경에서만 사용할 수 있습니다. 타사 서버에 배포된 AIDE 소프트웨어는 메타데이터 기능으로 제한되며 AIDE 라이센스가 필요하지 않습니다.



=== 라이센스 설치 요구 사항

NetApp DCN에서 라이선스가 필요한 AIDE 기능을 사용하는 배포 환경의 경우, AIDE 라이선스를 확보하고 관련 NLF 파일을 로컬 시스템에 다운로드해야 합니다. 그런 다음 System Manager를 통해 해당 파일을 AFX 스토리지 시스템에 업로드할 수 있습니다. 또한 다음 사항을 확인하십시오.

* ONTAP System Manager에 로그인하기 위한 관리자 자격 증명
* ONTAP 9.18.1 이상을 실행하는 AFX 클러스터




== AFX 시스템에 라이센스 설치

AFX 스토리지 시스템에 필요한 추가 AIDE 기능을 활성화하려면 AIDE 라이센스를 설치할 수 있습니다.

.단계
. System Manager에서 * 클러스터 * 를 선택한 다음 * 설정 * 을 선택합니다.
. *라이선스* 옆에서 image:icon_arrow.gif["오른쪽을 가리키는 파란색 화살표"]을(를) 선택합니다.
. 사용 가능한 ONTAP 기능을 표시하려면 *Features* 탭을 선택하십시오.
. 라이센스를 설치하려면 *설치된 라이센스* 탭을 선택하십시오.
. image:icon_add_blue_bg.png["파란색 사각형 안에 더하기 기호가 있고 그 뒤에 흰색 글자로 add라는 단어가 있습니다"]를 선택합니다.
. 로컬 라이선스 파일을 선택하고 *추가*를 선택합니다.
