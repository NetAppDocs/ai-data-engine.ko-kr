= NetApp DCN 배포 환경에서 AIDE 소프트웨어 빠른 시작 가이드
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp DCN 배포 환경에서 AI Data Engine(AIDE)을 사용하려면 AFX 및 NetApp DCN 하드웨어를 설치하고 클러스터를 설정한 다음 작업 공간을 생성해야 합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["하나"] AIDE를 새 AFX 클러스터에 설치하시겠습니까, 아니면 기존 AFX 클러스터에 설치하시겠습니까?
[role="quick-margin-para"]
AIDE와 AFX를 동시에 설치할지, 아니면 기존 AFX 클러스터에 AIDE를 통합할지 결정해야 합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["둘"] NetApp DCN 배포 요구 사항 검토
[role="quick-margin-para"]
시작하기 전에 하드웨어 및 소프트웨어 요구 사항을 검토하십시오. link:../install-dcn/aide-dcn-hardware-requirements.html["NetApp DCN 요구 사항에 대해 알아보세요"].

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["3"]AFX 및 NetApp DCN 하드웨어 설치 및 설정
[role="quick-margin-para"]
AFX 스토리지 시스템을 아직 설치하지 않은 경우 https://docs.netapp.com/us-en/ontap-afx/install-setup/install-setup-workflow.html["설치"^] AFX 하드웨어를 설치합니다.

[role="quick-margin-para"]
NetApp DCN 노드를 설치하고 설정하십시오. link:../install-dcn/install-workflow.html["NetApp DCN 노드를 설치하고 설정하세요."].

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["네"] 클러스터를 설정하세요
[role="quick-margin-para"]
ONTAP System Manager를 사용하여 link:../install-setup/cluster-setup-afx.html["AFX 클러스터를 사용하여 AIDE를 설정합니다"]에 대한 빠르고 간편한 프로세스를 안내받으세요.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["다섯"] 작업 공간 및 데이터 액세스 설정
[role="quick-margin-para"]
link:../workspaces/create-workspace.html["AIDE 데이터에 액세스할 수 있는 작업 공간 및 해당 작업 공간의 사용자를 설정합니다"].

.다음 단계
이제 ONTAP System Manager를 사용하여 AIDE를 관리하고 데이터 엔지니어 및 데이터 과학자가 작업 공간과 구성을 시작할 수 있습니다. 작업 공간 권한이 있는 사용자는 AIDE 라이선스 및 권한 수준에 따라 link:../vectorization/zero-to-rag.html["고급 AIDE 구성"] 작업을 수행할 수 있습니다.
