= AI Data Engine software 업데이트
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
스토리지 관리자는 ONTAP System Manager를 사용하여 AIDE 시스템의 AI Data Engine(AIDE) software, DCN(데이터 컴퓨팅 노드) 펌웨어 및 기타 시스템 파일을 업데이트할 수 있습니다.

AIDE는 ONTAP 자동 소프트웨어 업데이트 기능을 지원하지 않습니다. link:https://mysupport.netapp.com/site/downloads["NetApp Support 사이트 다운로드"^]에서 알림을 받도록 등록할 수 있지만 AIDE 소프트웨어의 모든 업데이트는 관리자가 수동으로 수행합니다.

.이 작업 정보
통합 AI Data Engine software 패키지는 일반적인 ONTAP 업데이트 패키지(AIDE 패키지는 약 40GB)보다 훨씬 큽니다. AIDE software를 업데이트할 때는 업로드 및 설치 시간이 더 오래 걸릴 수 있으므로 참고하시기 바랍니다.

.시작하기 전에
* DCN 펌웨어 및 AI Data Engine software를 업데이트하려면 스토리지 관리자 권한이 필요합니다.
* 활성 계정에 대한 NetApp Support Site 자격 증명이 필요합니다.
* link:upgrades-aide-summary.html#compatibility-matrix["업데이트 계획 시 ONTAP, DCN 펌웨어 및 AI Data Engine software 간의 호환성을 확인하십시오"].



IMPORTANT: DCN 펌웨어 또는 AI Data Engine software 업데이트의 경우 되돌리기 기능이 지원되지 않습니다. 업데이트 설치 후에는 이전 버전으로 되돌릴 수 없습니다.

.단계
. https://mysupport.netapp.com/site/products/all/details/aide/downloads-tab["결합된 DCN 펌웨어 및 AIDE 소프트웨어 업데이트 파일을 로컬 클라이언트에 다운로드합니다"^].
. System Manager에서 *클러스터 > 설정 > 소프트웨어 업데이트*를 선택합니다.
. *소프트웨어 업데이트* 옆에서 image:icon_arrow.gif["오른쪽을 가리키는 파란색 화살표"]을(를) 선택합니다.
. AI Data Engine 업데이트에서 *AI Data Engine software 파일 추가*를 선택하고 업데이트 패키지를 선택하십시오.
. 패키지 업로드가 완료되면 *업데이트*를 선택하여 DCN 노드에 업데이트 설치를 시작합니다.
+

NOTE: DCN 노드 업데이트가 진행 중일 때는 AI Data Engine Console을 사용할 수 없거나 액세스할 수 없습니다.



.결과
DCN은 AI Data Engine software로 업데이트되며 업데이트된 버전은 각 노드에 표시됩니다.
